## 🎯 Workshop Design Approach

This workshop uses **Backward Design** (Wiggins & McTighe):

1. **Desired Results** — Attendees can write a context-rich prompt and run a live Canvas task
2. **Acceptable Evidence** — They leave with one usable artifact
3. **Learning Experience** — 30 min: framing (5) → demo (20) → Q&A (5)
