What to Expect in This Course

🗓 How the Course Is Structured

Three modules build on each other. Each module unlocks after you complete the prior one.

  • Module 1 (Weeks 1–2): See It — Foundations of Data Visualization
  • Module 2 (Weeks 3–5): Say It — Building the Data Narrative
  • Module 3 (Weeks 6–7): Show It — Presenting to Decision Makers
