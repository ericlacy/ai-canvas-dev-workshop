BUS 3410  ·  7-Week Accelerated  ·  3 Credits

Data Storytelling for Business

Turn data into decisions. Three modules. One repeatable framework.

  • Read the Course Syllabus

  • Post in the Week 1 Discussion — share a visualization and evaluate it using the CLEAR rubric

Course Learning Outcomes

By the end of this course, students will be able to:

  • Select the appropriate chart type for a given data set and business question
  • Construct a data narrative that drives a clear, audience-specific recommendation
  • Design accessible, WCAG-compliant data visualizations using industry tools
  • Evaluate data stories for accuracy, bias, and persuasive integrity
  • Present data findings to a non-technical executive audience with confidence
7-Week Course Map

Module 1  ·  Weeks 1–2

See It: Foundations of Data Visualization

Current

Match chart types to data structures and business questions. Apply perception science to understand why some visuals communicate faster than others.

Module Objectives

  • Classify data types and select appropriate chart families
  • Apply Gestalt principles to evaluate visual clarity
  • Critique a published visualization using a structured rubric
